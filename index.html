Quem somos nós?

Somos uma empresa de serviços de bilhete de transporte público.

Trabalhamos com um sistema de recarga de bilhetes, em que cada usuário possui o seu bilhete único gerado no primeiro acesso. Esse código deve ser guardado pelo usuário para futuras utilizações, como recarga do bilhete, verificação de bilhetes ativos e outras ações.

Clique no botão abaixo para gerar o código do seu bilhete.

ATENÇÃO: LEMRE-SE DE ANOTAR O CÓDIGO MOSTRADO NA TELA. APÓS GERAR O BILHETE, VOCÊ DEVERÁ CONFIRMAR O CÓDIGO NO CAMPO APROPRIADO, INSERINDO O NÚMERO DO CÓDIGO.
